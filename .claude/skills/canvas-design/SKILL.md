# Canvas Design

Create visually sophisticated art in PNG and PDF formats using design philosophy as a foundation. This approach emphasizes original visual designs while avoiding copyright concerns.

## Two-Step Process

### Phase 1: Design Philosophy Creation

Develop an aesthetic movement expressed through form, space, color, and composition rather than lengthy text. The philosophy should:

- Feature a movement name (1-2 words)
- Articulate vision in 4-6 concise paragraphs
- Address space/form, color/material, scale/rhythm, composition, and hierarchy
- Emphasize craftsmanship REPEATEDLY with phrases like "meticulously crafted" and "painstaking attention"
- Keep text minimal, allowing creative interpretation

### Phase 2: Canvas Expression

Transform the philosophy into visual artifacts that are "90% visual design, 10% essential text." Key requirements include:

- Single-page, highly visual outputs (unless multi-page requested)
- Text as a contextual element using design-forward typography
- Museum or magazine-quality execution
- Professional spacing with no overlaps or elements extending beyond boundaries
- Appearance of extensive, expert-level labor

## Critical Design Principles

### Conceptual Depth
Embed subtle thematic references that reward familiar audiences while offering sophisticated abstract appeal to others.

### Minimalism
Information communicates through visual means—space, color, form—rather than explanatory text.

### Craftsmanship Standards
Every detail reflects expert execution worthy of museum display.

## Output Format

Deliverables include:
- `.md` philosophy file
- Visual expression as `.pdf` or `.png`

## Usage

Use this skill when creating:
- Visual design artifacts
- Art pieces with sophisticated aesthetics
- Museum-quality visual outputs
- Design philosophy documents
