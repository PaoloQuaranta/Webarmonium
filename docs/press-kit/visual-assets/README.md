## File naming convention

`{subject}-{state}-{dimension-or-duration}.{ext}`

Examples:
- `hero-1920.png` (subject: hero, dimension: 1920px wide)
- `room-detail-sequencer.png` (subject: room, state: detail-sequencer)
- `documentation-video-3min.mp4` (subject: documentation, duration: 3 min)
- `feed-detail-loop-15s.gif` (subject: feed, state: detail-loop, duration: 15 s)
