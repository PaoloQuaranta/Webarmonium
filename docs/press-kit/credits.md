## Live data sources

Webarmonium hears the public surface of the web through three open APIs. None of these sources participate in the work; their data is used under their public-API terms.

- **Wikipedia** — RecentChanges API (English Wikipedia)
- **Y Combinator's Hacker News** — Firebase API (`newstories`, `item`)
- **GitHub** — Public Events API

## Open-source dependencies

- [Tone.js](https://tonejs.github.io/) — Web Audio synthesis framework, by Yotam Mann and contributors
- [Socket.io](https://socket.io/) — real-time bidirectional communication, by Guillermo Rauch and contributors
- [PixiJS](https://pixijs.com/) — WebGL rendering engine
- [Express.js](https://expressjs.com/) — Node.js HTTP framework
- [standardized-audio-context](https://github.com/chrisguttandin/standardized-audio-context) — by Christoph Guttandin

Full dependency manifest in the source repository.
